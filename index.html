SQL.js • HTML • CSS

Conception et déploiement d'une application web de gestion de tâches sécurisée, sans backend ni serveur. Authentification par mot de passe hashé (PBKDF2) avec chiffrement AES-256 des données via l'API Web Crypto, stockage persistant côté client avec SQLite (sql.js) et OPFS.
